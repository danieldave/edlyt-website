How to use this pack

Each component includes HTML structure and CSS styling. Copy the HTML block and the corresponding CSS rules into your project. JS behavior (where needed) will be added in the next phase.

Starter tips

  • Extract CSS to a separate file and scope selectors (e.g., .cmp-portfolio) to avoid clashes.
  • Replace placeholder images with real assets (use same aspect ratio to avoid layout shift).
  • When adding JS, keep interactions simple and accessible (aria attributes, keyboard support).
